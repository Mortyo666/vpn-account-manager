# vpn-account-manager
Интерактивный скрипт для управления учётками OpenVPN и WireGuard на Linux

## Описание

Скрипт предоставляет интерактивное меню для управления учётными записями VPN:
- **OVPN**: Запуск существующего Python-скрипта для создания учёток OpenVPN
- **WireGuard**: Автоматическое создание конфигурации WireGuard с заданным количеством клиентов

## Требования

- Linux (тестировалось на CentOS 7)
- WireGuard (`wg-quick`)
- Python 3 (для OVPN скрипта)
- Root права для управления сетевыми интерфейсами

## Установка и запуск

### Способ 1: Прямой запуск с GitHub

```bash
curl -sL https://raw.githubusercontent.com/Mortyo666/vpn-account-manager/main/vpn_manager.sh | sudo bash
```

### Способ 2: Скачивание и запуск

```bash
# Скачать скрипт
curl -sL https://raw.githubusercontent.com/Mortyo666/vpn-account-manager/main/vpn_manager.sh -o vpn_manager.sh

# Сделать исполняемым
chmod +x vpn_manager.sh

# Запустить
sudo ./vpn_manager.sh
```

## Использование

При запуске скрипт предложит выбрать:

```
Выберите действие:
1) Создать учётки OVPN (+5)
2) Создать учётки WG
```

### Опция 1: OVPN
Запускает Python-скрипт `menu.py` из `/usr/local/openvpn_scripts/` для управления OpenVPN учётками.

### Опция 2: WireGuard
Предложит ввести количество учёток и автоматически:
1. Остановит интерфейс `wg00`
2. Создаст новую конфигурацию
3. Добавит клиентов с IP-адресами `10.0.0.2`, `10.0.0.3` и т.д.
4. Запустит интерфейс заново

## Примечания

- Перед использованием отредактируйте `<ваш_private_key>` и `ключ_$i` в скрипте на реальные ключи
- Убедитесь, что Python-скрипт `menu.py` существует по пути `/usr/local/openvpn_scripts/`
- Скрипт требует root-прав для управления сетевыми интерфейсами
